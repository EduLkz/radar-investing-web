# Radar Investing Web

O **Radar Investing Web** é uma solução de interface de usuário voltada para o monitoramento e análise inteligente de dados financeiros. Desenvolvido para oferecer agilidade na tomada de decisão, o sistema integra uma camada de visualização em React a um backend robusto em Python (Flask), permitindo o rastreamento em tempo real de ativos e tendências de mercado.

## 🎯 Principais Funcionalidades
* **Dashboard de Ativos:** Visualização centralizada de indicadores financeiros e histórico de performance.
* **Integração de Backend:** Comunicação otimizada com API via Flask, garantindo performance no processamento de dados.

## 🛠 Stack Tecnológica

### Frontend
* **React.js:** Interface baseada em componentes para maior reatividade e facilidade de manutenção.
* **React Router:** Gerenciamento de rotas para navegação dinâmica entre as diferentes áreas do sistema.
* **Axios:** Consumo de dados da API via requisições assíncronas.

### Backend
* **Python (Flask):** Framework leve e eficiente para servir a API de dados.
* **SQLAlchemy:** ORM utilizado para a interação segura e organizada com o banco de dados.
